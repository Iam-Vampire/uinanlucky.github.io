## CSS Grid 布局指南

CSS Grid 是强大的二维布局系统。

### 基本概念

- Grid Container — 网格容器
- Grid Item — 网格项目
- Grid Line — 网格线
- Grid Cell — 网格单元

### 示例

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 16px;
}
```
