# Welcome to WikiWarm

这是一个融合了 **Wikitten** 的 Wiki 结构 + **Warmpaper** 温暖风格的 Hexo 主题。

## 特性

- **多级分类侧边栏** — 支持 3 层嵌套，点击展开/收起，手风琴式交互
- **温暖配色** — Claude 灵感米色背景 + 橙色强调 + 方格草稿纸纹理
- **霞鹜文楷** — 全站 LXGW WenKai GB 字体，优雅阅读体验
- **深色模式** — 支持自动/手动切换
- **TOC 目录** — 文章右侧 sticky 目录，滚动高亮

## 分类结构

侧边栏显示为嵌套的树形结构，点击分类名展开/收起。

```
Tech/                          Tech
├── Tech Overview              ├── Frontend/
├── Frontend/                  │   ├── React 指南
│   ├── React 指南             │   ├── Vue 指南
│   ├── Vue 指南               │   └── CSS 布局/
│   └── CSS 布局/              │       ├── Flexbox
│       ├── Flexbox 指南       │       └── Grid
│       └── Grid 指南          └── Backend/
├── Backend/                       ├── Node.js
│   ├── Node.js 指南              ├── Python
│   ├── Python 指南               └── Database/
│   └── Database/                     ├── MongoDB
│       ├── MongoDB 指南             └── Redis
│       └── Redis 指南           └── DevOps/
└── DevOps/                          ├── Docker
    ├── Docker 指南                 └── CI-CD/
    └── CI-CD/                          ├── GitHub Actions
        ├── GitHub Actions             └── Jenkins
        └── Jenkins

Design/                          Design/
├── UI 设计原则                  ├── UI 设计

```

支持 N 层嵌套，当前最高 4 层（Tech > DevOps > CI-CD > Jenkins）。
